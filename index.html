# PRESENTING CODE

So far we have learned about the bare minimum structural tags necessary for every HTML document:

  1. <!DOCTYPE html>

  2. <html></html>: Indicates the start of an HTML document.

  3. <head></head>: Contains the metadata about the web page, such as the title, description, and keywords.

  4. <body></body>: Contains the visible content of the web page, such as text, images, videos, and other media.

  5. <footer></footer>: Contains info about the author, copyright, contact info, and social media links.
